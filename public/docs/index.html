Introduction

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

curl --request POST \
    "http://127.0.0.1:8000/api/customer/update/profile" \

    --data "{
    \"name\": \"accusamus\",
    \"phone\": \"suscipit\",
    \"birthday\": \"2023-06-11T02:07:32\"
}"

const url = new URL(

);

};

let body = {
    "name": "accusamus",
    "phone": "suscipit",
    "birthday": "2023-06-11T02:07:32"
};

POST api/search/service

Burada hizmet türüne veya şehir verisine göre işletmelerin verisi döndürülür. Not: city_id ve service_id ikisinden sadece biri de gönderilebilir.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/search/service?service_id=illo&city_id=harum"
const url = new URL(
    "http://127.0.0.1:8000/api/search/service"
);

const params = {
    "service_id": "illo",
    "city_id": "harum",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

fetch(url, {
    method: "POST",
}).then(response => response.json());
